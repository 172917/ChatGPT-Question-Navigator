## Title

ChatGPT Question Navigator: Organize, Search & Revisit Long Chats｜ChatGPT 問題導航：自動整理長對話

## Description

ChatGPT Question Navigator is a local-first Chrome / Edge extension that turns questions in long ChatGPT conversations into a searchable working index.

It automatically organizes questions into Code, Solutions, Research, To-dos, and Other. Search question text and quotes, favorite important turns, correct categories when needed, switch between Simplified Chinese and English, and jump back to the original message instantly.

GitHub:
https://github.com/172917/ChatGPT-Question-Navigator

Key features:
- One-click jump back to earlier questions
- Local automatic categories: Code, Solutions, Research, To-dos, and Other
- Search across question text and quoted snippets
- Per-conversation favorites and manual category corrections
- Shows quoted snippets with related questions
- Simplified Chinese / English interface with automatic language detection
- Floating panel with drag, resize, and collapse controls
- Local-first design: no chat upload, no remote storage
- GPL-3.0 open source
